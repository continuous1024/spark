## Spark Core 作业执行流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant SC as SparkContext
    participant DAGScheduler as DAG调度器
    participant TaskScheduler as 任务调度器
    participant Worker as 工作节点
    participant Executor as 执行器

    Client->>SC: 创建RDD并调用行动操作
    SC->>DAGScheduler: 提交作业(Job)

    DAGScheduler->>DAGScheduler: 创建逻辑执行计划
    DAGScheduler->>DAGScheduler: 将作业划分为多个阶段(Stage)
    Note over DAGScheduler: 根据shuffle依赖划分阶段

    loop 对每个阶段
        DAGScheduler->>TaskScheduler: 提交任务集(TaskSet)
        TaskScheduler->>Executor: 分发任务(Task)
        Executor->>Executor: 执行任务

        alt 任务成功
            Executor->>TaskScheduler: 返回任务结果
        else 任务失败
            Executor->>TaskScheduler: 报告任务失败
            TaskScheduler->>TaskScheduler: 重试任务(最多3次)
        end
    end

    TaskScheduler->>DAGScheduler: 返回阶段结果
    DAGScheduler->>SC: 返回作业结果
    SC->>Client: 返回最终结果
```

## Spark SQL 查询执行流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Session as SparkSession
    participant Analyzer as 分析器
    participant Optimizer as 优化器
    participant Planner as 物理计划生成器
    participant Executor as 执行引擎

    Client->>Session: 提交SQL查询或DataFrame操作
    Session->>Analyzer: 解析SQL/DataFrame为未解析逻辑计划

    Analyzer->>Analyzer: 应用分析规则
    Note over Analyzer: 解析表名、列名、函数等
    Analyzer->>Optimizer: 生成已解析逻辑计划

    Optimizer->>Optimizer: 应用优化规则
    Note over Optimizer: 谓词下推、列剪裁、常量折叠等
    Optimizer->>Planner: 生成优化后的逻辑计划

    Planner->>Planner: 生成多个物理计划
    Planner->>Planner: 使用成本模型选择最优计划
    Planner->>Executor: 生成RDD执行计划

    Executor->>Executor: 执行查询
    Note over Executor: 可能使用全代码生成优化
    Executor->>Session: 返回查询结果
    Session->>Client: 返回最终结果
```

## Structured Streaming 流处理流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Session as SparkSession
    participant StreamExec as StreamExecution
    participant Source as 数据源
    participant Trigger as 触发器
    participant Processor as 微批处理器
    participant StateStore as 状态存储
    participant Sink as 输出接收器

    Client->>Session: 创建流式DataFrame并定义查询
    Session->>StreamExec: 创建流执行引擎
    StreamExec->>Source: 初始化数据源
    StreamExec->>Sink: 初始化接收器
    StreamExec->>StateStore: 初始化状态存储

    loop 流处理循环
        Trigger->>StreamExec: 触发新的微批次
        StreamExec->>Source: 获取最新偏移量
        Source->>StreamExec: 返回可用数据的偏移量

        alt 有新数据
            StreamExec->>Source: 获取偏移量范围内的数据
            Source->>StreamExec: 返回批次数据
            StreamExec->>Processor: 处理批次数据

            Processor->>StateStore: 读取状态(如有)
            Processor->>Processor: 执行查询逻辑
            Processor->>StateStore: 更新状态(如有)

            Processor->>StreamExec: 返回处理结果
            StreamExec->>Sink: 输出结果
            StreamExec->>StreamExec: 提交偏移量
        else 无新数据
            Note over StreamExec: 等待下一次触发
        end
    end

    Client->>StreamExec: 停止查询
    StreamExec->>Source: 停止数据源
    StreamExec->>Sink: 停止接收器
```

## Spark Streaming 流处理流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Context as StreamingContext
    participant Receiver as 接收器
    participant ReceiverSupervisor as 接收器监督器
    participant JobScheduler as 作业调度器
    participant JobGenerator as 作业生成器
    participant DStreamGraph as DStream图
    participant Executor as 执行器

    Client->>Context: 创建DStream并定义转换
    Context->>DStreamGraph: 构建DStream依赖图
    Context->>Receiver: 启动接收器
    Receiver->>ReceiverSupervisor: 注册接收器

    loop 流处理循环
        Note over JobGenerator: 按批处理间隔触发
        JobGenerator->>ReceiverSupervisor: 获取接收到的数据
        ReceiverSupervisor->>JobGenerator: 返回接收到的数据块

        JobGenerator->>DStreamGraph: 生成RDD DAG
        JobGenerator->>JobScheduler: 提交批处理作业

        JobScheduler->>Executor: 执行批处理作业
        Executor->>JobScheduler: 返回批处理结果

        alt 输出操作
            JobScheduler->>Executor: 执行输出操作
        end

        JobGenerator->>JobGenerator: 更新检查点(如启用)
    end

    Client->>Context: 停止流处理
    Context->>JobGenerator: 停止作业生成
    Context->>Receiver: 停止接收器
```

## MLlib 机器学习流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Session as SparkSession
    participant Pipeline as 机器学习管道
    participant Transformer as 转换器
    participant Estimator as 估计器
    participant Evaluator as 评估器
    participant Model as 模型

    Client->>Session: 加载训练数据
    Client->>Pipeline: 创建机器学习管道

    loop 添加管道阶段
        alt 数据转换
            Client->>Transformer: 创建转换器(如特征工程)
            Client->>Pipeline: 添加转换器到管道
        else 模型训练
            Client->>Estimator: 创建估计器(如算法)
            Client->>Pipeline: 添加估计器到管道
        end
    end

    Client->>Pipeline: 拟合训练数据
    Pipeline->>Pipeline: 执行管道各阶段

    loop 对每个管道阶段
        alt 转换器阶段
            Pipeline->>Transformer: 应用转换
            Transformer->>Pipeline: 返回转换后的数据
        else 估计器阶段
            Pipeline->>Estimator: 拟合数据
            Estimator->>Model: 生成模型
            Model->>Pipeline: 返回训练好的模型
        end
    end

    Pipeline->>Client: 返回管道模型

    Client->>Session: 加载测试数据
    Client->>Model: 应用模型进行预测
    Model->>Client: 返回预测结果

    Client->>Evaluator: 评估模型性能
    Evaluator->>Client: 返回评估指标
```

## GraphX 图计算流程

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant SC as SparkContext
    participant GraphLoader as 图加载器
    participant Graph as 图
    participant VertexRDD as 顶点RDD
    participant EdgeRDD as 边RDD
    participant Pregel as Pregel引擎

    Client->>SC: 创建SparkContext
    Client->>GraphLoader: 加载图数据
    GraphLoader->>VertexRDD: 创建顶点RDD
    GraphLoader->>EdgeRDD: 创建边RDD
    GraphLoader->>Graph: 构建图
    GraphLoader->>Client: 返回图对象

    alt 图转换操作
        Client->>Graph: 调用转换操作(如mapVertices)
        Graph->>Graph: 执行转换
        Graph->>Client: 返回新图
    else 图聚合操作
        Client->>Graph: 调用聚合操作(如aggregateMessages)
        Graph->>Graph: 执行消息传递和聚合
        Graph->>Client: 返回聚合结果
    else 迭代图计算
        Client->>Pregel: 调用Pregel API

        loop 迭代计算
            Pregel->>Graph: 发送消息
            Graph->>Graph: 更新顶点状态
            Graph->>Pregel: 检查终止条件
        end

        Pregel->>Client: 返回计算结果
    end

    Client->>Graph: 获取结果(如顶点或边属性)
    Graph->>Client: 返回计算结果
```
