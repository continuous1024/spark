# Apache Spark 数据流向图

## RDD 数据流

```
+----------------+     +----------------+     +----------------+
|  数据源         |     |  转换操作       |     |  行动操作       |
| (HDFS, S3等)   | --> | (map, filter等) | --> | (count, save等) |
+----------------+     +----------------+     +----------------+
```

详细流程：

```
+----------------+     +----------------+     +----------------+     +----------------+
|                |     |                |     |                |     |                |
|   数据源        | --> |   分区 RDD     | --> |  转换后的 RDD   | --> |  结果 RDD      |
|                |     |                |     |                |     |                |
+----------------+     +----------------+     +----------------+     +----------------+
                                |                    |                      |
                                v                    v                      v
                       +----------------+   +----------------+    +----------------+
                       |                |   |                |    |                |
                       |   Task 1       |   |   Task 2       |    |   Task N       |
                       |                |   |                |    |                |
                       +----------------+   +----------------+    +----------------+
                                |                    |                      |
                                v                    v                      v
                       +----------------+   +----------------+    +----------------+
                       |                |   |                |    |                |
                       |   Executor 1   |   |   Executor 2   |    |   Executor N   |
                       |                |   |                |    |                |
                       +----------------+   +----------------+    +----------------+
```

## Spark SQL 查询执行流程

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  SQL / DataFrame| --> |  逻辑计划      | --> |  优化逻辑计划   |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
        |                                              |
        v                                              v
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  物理计划       | <-- |  优化物理计划   | <-- |  物理计划选择   |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
        |
        v
+----------------+     +----------------+
|                |     |                |
|  RDD 执行       | --> |  结果返回      |
|                |     |                |
+----------------+     +----------------+
```

详细流程：

```
+----------------+
|                |
|  SQL 查询       |
|                |
+----------------+
        |
        v
+----------------+
|                |
|  解析器         |
|                |
+----------------+
        |
        v
+----------------+
|                |
|  分析器         |
|                |
+----------------+
        |
        v
+----------------+     +----------------+
|                |     |                |
|  逻辑优化器     | --> |  优化逻辑计划   |
|                |     |                |
+----------------+     +----------------+
        |
        v
+----------------+     +----------------+
|                |     |                |
|  物理计划生成器  | --> |  物理计划候选   |
|                |     |                |
+----------------+     +----------------+
        |
        v
+----------------+     +----------------+
|                |     |                |
|  成本模型       | --> |  最优物理计划   |
|                |     |                |
+----------------+     +----------------+
        |
        v
+----------------+
|                |
|  代码生成       |
|                |
+----------------+
        |
        v
+----------------+
|                |
|  RDD 执行       |
|                |
+----------------+
```

## Structured Streaming 数据流

```
+----------------+     +----------------+     +----------------+     +----------------+
|                |     |                |     |                |     |                |
|  数据源         | --> |  输入表(无界)   | --> |  结果表(无界)   | --> |  输出          |
|                |     |                |     |                |     |                |
+----------------+     +----------------+     +----------------+     +----------------+
```

详细流程：

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  Source        | --> |  触发器        | --> |  微批次执行     |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
        |                                              |
        v                                              v
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  偏移量管理     | <-- |  检查点        | <-- |  状态管理      |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                                      |
                                                      v
                                              +----------------+
                                              |                |
                                              |  Sink          |
                                              |                |
                                              +----------------+
```

## Spark Streaming 数据流

```
+----------------+     +----------------+     +----------------+     +----------------+
|                |     |                |     |                |     |                |
|  数据源         | --> |  Receiver      | --> |  DStream       | --> |  输出操作      |
|                |     |                |     |                |     |                |
+----------------+     +----------------+     +----------------+     +----------------+
```

详细流程：

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  数据源         | --> |  Receiver      | --> |  接收器内存    |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                                      |
                                                      v
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  批处理作业     | <-- |  作业生成器     | <-- |  DStream       |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
        |
        v
+----------------+     +----------------+
|                |     |                |
|  RDD 处理       | --> |  输出操作      |
|                |     |                |
+----------------+     +----------------+
```

## MLlib 数据流

```
+----------------+     +----------------+     +----------------+     +----------------+
|                |     |                |     |                |     |                |
|  训练数据       | --> |  特征转换      | --> |  模型训练      | --> |  模型评估      |
|                |     |                |     |                |     |                |
+----------------+     +----------------+     +----------------+     +----------------+
        |                                                                     |
        |                                                                     v
        |                                                            +----------------+
        |                                                            |                |
        +------------------------------------------------------------> 模型应用       |
                                                                     |                |
                                                                     +----------------+
```

详细流程：

```
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  原始数据       | --> |  DataFrame     | --> |  特征工程      |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                                      |
                                                      v
+----------------+     +----------------+     +----------------+
|                |     |                |     |                |
|  模型服务       | <-- |  模型持久化     | <-- |  Pipeline     |
|                |     |                |     |                |
+----------------+     +----------------+     +----------------+
                                                      |
                                                      v
                                              +----------------+
                                              |                |
                                              |  模型评估      |
                                              |                |
                                              +----------------+
```
